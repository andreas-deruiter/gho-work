What should excludedTools control?

The codebase has two distinct mechanisms. Let's clarify which one(s) the UI should cover.

A

SDK Tools Only

Control which tools the Copilot SDK agent can use (bash, file_read, file_write, etc.). Power-user setting. Per-skill toggles are a separate feature.

B

Skills Only

Enable/disable individual skills (markdown system prompts) from the skill browser. The SDK tool list stays untouched.

C

Both — Unified UI

One settings page that lets users control both SDK tools and skills. Two sections on the same page.

D

Both — Separate Pages

Skills toggles on the existing Skills page. SDK tools on a new "Agent" or "Tools" settings page.
