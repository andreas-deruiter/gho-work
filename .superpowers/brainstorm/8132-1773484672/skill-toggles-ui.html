Key details

  • Toggle switch replaces the plain source label on the right side
  • Source badge (bundled/user) stays, moves left of the toggle
  • Disabled skills get opacity: 0.5 on the whole row
  • Toggle uses role="switch" + aria-checked for accessibility
  • Changes persist immediately via IPC (no save button)
