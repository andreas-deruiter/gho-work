How should the install flow work?

What happens after the user clicks "Install" on a plugin

A

Instant install — configure later

Click Install → plugin downloads in background → skills are immediately available. MCP servers that need configuration (API keys, etc.) show a "Configure" badge — user sets them up when ready. Fast, non-blocking.
