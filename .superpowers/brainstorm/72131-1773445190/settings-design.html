Data Flow

How settings are read and persisted
Appearance:
User clicks theme card → SettingsPanel calls IThemeService.setTheme()
ThemeService applies data-theme attr + persists via IPC → IStorageService.setSetting('theme', value)
On load: ThemeService reads stored theme via IPC → applies on startup
Skills:
SettingsPanel requests skill list via IPC → main process calls ISkillRegistry.list()
User adds path → IPC → IStorageService.setSetting('skills.additionalPaths', JSON.stringify([...]))
Main process calls ISkillRegistry.refresh() → emits onDidChangeSkills
SettingsPanel listens via IPC event → re-renders skill list
