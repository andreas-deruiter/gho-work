RUNTIME EVENTS
SubagentProgressBridge
plan_changed → parse checklist
  → InfoPanelState.setPlan()
subagent.started → mark step active
subagent.completed → mark step done
subagent.failed → mark step failed
  → ProgressSection re-renders

Message priority (top = highest)

1. SDK defaults (security, tool instructions) — we don't control this
2. gho-instructions.md (persona, planning, delegation) — bundled
3. User instructions (~/.gho/instructions.md) — user-level
4. Project instructions (GHO.md / CLAUDE.md / etc.) — project-level
5. context.systemPrompt (per-conversation, e.g. setup skills) — ephemeral

Note: systemMessage mode is 'append' — all content stacks after SDK defaults. Within our content, later entries can override earlier ones (project overrides user).
